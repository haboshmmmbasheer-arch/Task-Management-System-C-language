# Task Management System using C

## Description
This project implements a task management system using linked lists, stacks, and queues.

## Features
- Load tasks from file
- Add new tasks
- Delete tasks
- Search tasks by ID or name
- Perform tasks
- Undo last performed task
- View performed and unperformed tasks
- Generate summary report

## Data Structures Used
- Linked List
- Stack
- Queue

## Files
- main.c → program code
- tasks.txt → input file
- Report.txt → output report
